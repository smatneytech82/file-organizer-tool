# File Organizer Tool

## Overview
A Python automation utility that organizes files in the Downloads folder into categorized directories.

## Features
- Organizes files automatically
- Categorizes by file extension
- Creates folders dynamically
- Supports multiple file types

## Tech Stack
- Python
- os module
- shutil module

## How to Run
1. Update the downloads folder path if needed
2. Run organizer.py
3. Files will automatically organize into folders
